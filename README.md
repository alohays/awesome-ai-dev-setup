## CLI Scaffolding Patterns

Tools with UX and architecture patterns worth learning from.

### Modern Scaffolding Frameworks

- [Vercel/turborepo](https://github.com/vercel/turborepo) - Zero-config TypeScript generators using existing workspace as template. ![GitHub Stars](https://img.shields.io/github/stars/vercel/turborepo?style=flat-square)
- [t3-oss/create-t3-app](https://github.com/t3-oss/create-t3-app) - Interactive step-by-step prompts with modular optionality. ![GitHub Stars](https://img.shields.io/github/stars/t3-oss/create-t3-app?style=flat-square)
- [nrwl/nx](https://github.com/nrwl/nx) - Plugin-based generators with custom workspace generators in TypeScript. ![GitHub Stars](https://img.shields.io/github/stars/nrwl/nx?style=flat-square)
- [oclif/oclif](https://github.com/oclif/oclif) - Lifecycle hooks and auto-generated help used by Salesforce and Heroku CLI. ![GitHub Stars](https://img.shields.io/github/stars/oclif/oclif?style=flat-square)

### Micro-Generators

- [plopjs/plop](https://github.com/plopjs/plop) - Action-based generation with add, modify, and append for consistency. ![GitHub Stars](https://img.shields.io/github/stars/plopjs/plop?style=flat-square)
- [jondot/hygen](https://github.com/jondot/hygen) - Generators that live in your project, versioned and co-located. ![GitHub Stars](https://img.shields.io/github/stars/jondot/hygen?style=flat-square)
- [copier-org/copier](https://github.com/copier-org/copier) - Template evolution that updates projects when templates improve. ![GitHub Stars](https://img.shields.io/github/stars/copier-org/copier?style=flat-square)

### Dashboard-Specific

- [jakubplichta/grafana-dashboard-builder](https://github.com/jakubplichta/grafana-dashboard-builder) - YAML-to-JSON dashboard generation for dashboard-as-code. ![GitHub Stars](https://img.shields.io/github/stars/jakubplichta/grafana-dashboard-builder?style=flat-square)
